# Vim

The best configurations.

## Install Vundle

- ``` git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim ```

## Install Plugins with Vundle

- ``` git clone https://github.com/danvincenzi/vim.git ```

- ``` mv vim/.vimrc ~/ ```

- Inside the Vim: ``` :PluginInstall ```
